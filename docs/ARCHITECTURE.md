## Key Components

### Views
- `HomeView.swift` - Main upload screen
- `SplitScreenEditorView.swift` - Split-screen PDF editor

### ViewModels
- `HomeViewModel.swift` - Upload and processing logic
- `DocumentViewModel.swift` - Form editing and PDF sync

### Services
- `APIService.swift` - Backend API calls
- `LocalFormStorageService.swift` - Form data persistence
- `DocumentPickerService.swift` - File selection

### UI Components
- `PDFKitRepresentedView.swift` - PDFKit wrapper with annotations
- `AnimatedGradientBackground.swift` - Gradient animation

## Hybrid Detection Pipeline

### Step 1: AcroForm Detection (Primary)
- Detects native PDF form fields
- 100% accurate coordinates
- Fast (~100ms)

### Step 2: QuickLook Fallback
- For PDFs without AcroForm fields
- Uses Apple's ML form detector
- Same experience as Files.app

### Step 3: Synthetic Widgets (Optional)
- Creates interactive fields from OCR regions
- For scanned documents
